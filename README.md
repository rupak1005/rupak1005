## Architecture & Philosophy

Shipping blazingly fast, highly concurrent web applications. Obsessed with reducing cumulative layout shifts, optimizing WebGL render loops, and authoring zero-cost abstractions for the client side. 

- **Core Focus:** Architecting full-stack distributed systems and orchestrating fluid, hardware-accelerated user interfaces.
- **Graphics & DOM:** Leveraging Three.js and GSAP to write custom WebGL shaders and complex DOM animation timelines without blocking the main thread.
- **Compute:** Grinding DSA in C++ to write cache-friendly, time/space optimized algorithms.
- **Backend Infrastructure:** Authoring strongly-typed RESTful APIs and deploying serverless microservices utilizing Node, Python, and Go.
- **DevOps & CI/CD:** Containerizing with Docker, managing deployment pipelines via Vercel/Git, and daily-driving Arch Linux.
- **Open Source:** Upstreaming patches and building modular, accessible component libraries (Aceternity UI, Skiper UI).
